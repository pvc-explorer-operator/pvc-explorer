# Core Concepts

PVC Explorer is built around scoped, on-demand access to PVC data.

## Concepts

- Scope: policy and discovery boundary for managed explorers.
- Explorer: resource that exposes a PVC for interactive access.
- Lifecycle: connect, idle countdown, sleep, and reconnect.
- Access controls: auth roles plus scope-level constraints.
