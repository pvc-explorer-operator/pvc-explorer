# How-to: Connect to PVC

Connect to an explorer and open the file browser.

## Steps

1. Open explorer list at `/`.
2. Select the target explorer.
3. If needed, click Connect (wake).
4. Click Browse Files to open `/explorers/:ns/:name/files`.
5. Keep session active with heartbeat when idle warning appears.
