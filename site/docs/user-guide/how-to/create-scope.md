# How-to: Create Scope

Use the UI form to generate a `PVCExplorerScope` manifest.

## Steps

1. Open `/scopes/create`.
2. Enter scope identity and namespace selection.
3. Configure discovery mode and defaults.
4. Review generated YAML.
5. Copy or download and apply with `kubectl apply -f`.
