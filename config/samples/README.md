# Sample Resources

This directory contains example custom resources for local testing and demos.

## Files

- `pvcexplorer_v1alpha1_pvcexplorerscope.yaml`: sample `PVCExplorerScope`
- `pvcexplorer_v1alpha1_pvcexplorer.yaml`: sample `PVCExplorer`

## Apply samples

```bash
kubectl apply -k config/samples/
```

For installation and CRD setup details, see `../../docs/operations/crd-installation.md`.
